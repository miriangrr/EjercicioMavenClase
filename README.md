## PROYECTO INVENTARIO RPG 
Descripción del Proyecto
Este proyecto es un sistema de inventario para un juego de rol (RPG) desarrollado en Java. El sistema permite a los jugadores gestionar sus objetos, armas, armaduras y otros elementos de su inventario de manera eficiente. El proyecto incluye funcionalidades para agregar, eliminar y visualizar los objetos en el inventario, así como para equipar y desequipar armas y armaduras.

Requisitos
Java 25
Maven
Instalación
Clonar el repositorio

Ejecutar el comando mvn clean install para compilar el proyecto

Ejecutar el comando mvn exec:java para iniciar la aplicación

o Ejecutar el archivo .jar generado en la carpeta resources
